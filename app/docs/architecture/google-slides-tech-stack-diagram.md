## Step-by-Step Instructions for Google Slides

### Step 1: Create New Slide
1. Insert → New Slide → Blank
2. Set slide background to white or light gray

### Step 2: Add Title
1. Insert → Text box
2. Type: "What We'll Build: Our Tech Stack"
3. Font: 44pt, Bold
4. Center align
5. Position at top of slide

### Step 3: Add Each Technology (Vertical Layout)

**For Next.js:**
1. Insert → Shape → Rounded rectangle (or circle)
2. Size: 100x100px
3. Fill color: Black (#000000)
4. Add text "N" (for Next.js) in white, 48pt
   - OR download Next.js logo from their website
5. Insert text box to the right:
   - Line 1: "Next.js 14" (32pt, Bold)
   - Line 2: "with App Router" (20pt, Regular, Gray)

**For Neon:**
1. Shape: 100x100px rounded rectangle
2. Fill: Neon green (#00E699)
3. Text/Logo: "NEON" in white
4. Right text box:
   - Line 1: "Neon PostgreSQL" (32pt)
   - Line 2: "Cloud Database" (20pt)

**For Prisma:**
1. Shape: 100x100px rounded rectangle
2. Fill: Dark blue-gray (#2D3748)
3. Logo: Prisma triangle (download from prisma.io)
4. Right text box:
   - Line 1: "Prisma ORM" (32pt)
   - Line 2: "Type-Safe Queries" (20pt)

**For shadcn/ui:**
1. Shape: 100x100px rounded rectangle
2. Fill: Near black (#09090b)
3. Text: "UI" in white
4. Right text box:
   - Line 1: "shadcn/ui" (32pt)
   - Line 2: "Beautiful Components" (20pt)

**For Git:**
1. Shape: 100x100px rounded rectangle
2. Fill: Git orange (#F05032)
3. Logo: Git logo (download from git-scm.com)
4. Right text box:
   - Line 1: "Git Repository" (32pt)
   - Line 2: "Version Control" (20pt)

### Step 4: Add Connecting Arrows
1. Insert → Line → Arrow
2. Style: Straight down arrow
3. Color: Blue (#3b82f6)
4. Weight: 6pt
5. Place between each technology

### Step 5: Align Everything
1. Select all elements
2. Format → Align → Center horizontally
3. Distribute vertically with even spacing

## Pro Tips

1. **Use Consistent Spacing:** 
   - All icons same size (100x100px)
   - All text boxes aligned vertically
   - Equal spacing between items (60px)

2. **Color Coordination:**
   - Use official brand colors
   - Keep text readable (dark on light, light on dark)

3. **Font Consistency:**
   - Title: Bold, 44pt
   - Tech names: Bold, 32pt  
   - Descriptions: Regular, 20pt

4. **Visual Hierarchy:**
   - Largest: Title
   - Medium: Technology names
   - Smallest: Descriptions

5. **Keep It Clean:**
   - Don't overcrowd
   - Plenty of white space
   - Easy to read from distance
